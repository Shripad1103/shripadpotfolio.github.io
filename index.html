Face Recognition Website

I developed a dynamic and responsive website using React, which performs face detection on images provided via a URL, leveraging the Clarifai API. This project showcases my expertise in frontend development, server-side programming, and database management. Key features include user authentication, allowing new users to sign up and existing users to log in seamlessly, and face detection, where users can input image URLs to detect faces within these images. Additionally, the website tracks the number of times a user logs in, maintaining a record of each session. For the backend, I utilized Node.js to handle server requests and PostgreSQL for robust and scalable database management. This project effectively combines my skills in React, Node.js, and PostgreSQL to deliver a full-stack application with practical, real-world utility.
